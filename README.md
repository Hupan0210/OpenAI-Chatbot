# OpenAI-Chatbot

一个基于 OpenAI GPT-4 API 的开源聊天机器人，致力于为中文用户提供高质量的 AI 对话体验。

## 项目特性

- 支持多平台接入（Web、微信、QQ 等）
- 丰富的 API 接口，便于二次开发
- 支持自定义指令和插件扩展
- 简单易用，部署方便

## 快速开始

1. 安装依赖

   ```bash
   pip install -r requirements.txt
   ```

2. 运行主程序

   ```bash
   python app/main.py
   ```

## 目录结构
